Requisitos para el icono de la aplicación:

1. Archivo: icon.png
2. Ubicación: o16_field_management/static/description/icon.png
3. Dimensiones recomendadas: 140x140 píxeles
4. Formato: PNG con fondo transparente
5. Estilo sugerido:
   - Diseño minimalista
   - Colores corporativos
   - Símbolo relacionado con canchas de fútbol
   - Bordes redondeados si se usa un fondo sólido

Sugerencias de diseño:
- Campo de fútbol estilizado
- Balón de fútbol moderno
- Combinación de verde (campos) y blanco
- Usar gradientes sutiles para dar profundidad

El icono aparecerá en:
1. Menú de aplicaciones de Odoo
2. Listado de módulos
3. Esquina superior izquierda cuando la aplicación está activa
